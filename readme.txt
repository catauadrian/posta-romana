This project is a wordpress plugin for Romanian Post Office documents, based on datatables library.

Plugin Name : Posta-romana

The steps involved are: 

1) Copy the files into plugin wordpress folder

2) Create some 3 extra tables into your woocommerce database.

wp_tblanul
wp_tbl_months
wp_company

using mysqlworkbench and import cvs files. See the in the picture: import-data-mysql.png
